# 디자인 레퍼런스 — hospital-homepage-xi.vercel.app (리엔느 피부과)

작업순서 3단계: 이 사이트의 디자인 요소를 TED 정보보호 블로그에 참고 적용.

## 기술 스택
- Next.js + Tailwind CSS

## 폰트
- 본문/제목 공통: **Pretendard Variable / Pretendard** (system-ui fallback)

## 스타일 특징
- 부드러운 그라데이션: 화이트 → 라이트블루 (`#fff` → `#e8f3fa`, rgba 50~70% 투명도 활용)
- 글래스모피즘: `backdrop-blur` 8px/12px (헤더·카드)
- 대형 블러 글로우: `blur(40px)`, `blur(64px)` 배경 장식
- 전체적으로 청결·세련·프리미엄 클리닉 무드 (화이트+블루+골드 포인트)

## 섹션 구성 (참고)
1. 글래스 네비게이션 헤더
2. 히어로 (대형 슬로건 + CTA)
3. 시그니처 서비스 캐러셀
4. 브랜드 철학 섹션
5. 이벤트/혜택 그리드
6. 숫자 통계 (경력 연차, 누적 건수, 만족도 %)
7. 오시는 길 + 상담 폼
8. 다크 네이비 푸터
